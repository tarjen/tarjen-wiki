# 关于

这是一个个人编程比赛记录 wiki。

## 用了什么

- [MkDocs](https://www.mkdocs.org/) + [Material 主题](https://squidfunk.github.io/mkdocs-material/)
- 托管在 GitHub Pages
- 内容全部用 markdown 维护

## 怎么贡献 / 修改

1. 改文件
2. `mkdocs gh-deploy` 发布

也可以直接在 GitHub 网页上点「编辑此页」按钮（每页右上角）。

## 图例

- `O` = 通过
- `.` = 未通过或未尝试
- `!` = 赛中未通过，赛后补题通过
- `?` = 待补
