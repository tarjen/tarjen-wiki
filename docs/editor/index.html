🍪 QOJ Cookie（导入比赛需要）

编辑器直接在浏览器里 fetch(qoj.ac/...)，需要带你的 QOJ 登录 cookie。 只存浏览器 localStorage，不上传服务器；过期了（一般 7–30 天）来改一次。

取法：浏览器登录 qoj.ac → F12 → Application → Cookies → qoj.ac → 选中那 3 行（uoj_remember_token / uoj_remember_token_checksum / UOJSESSID）→ 全选 Value 拼成 uoj_remember_token=VAL1;uoj_remember_token_checksum=VAL2;UOJSESSID=VAL3 粘到下面。

⚠️ CORS：qoj.ac 没设 CORS 头，浏览器会拒掉跨域 response。 需要装 Allow CORS 类扩展，对 qoj.ac enable。装完刷新本页面。
